# Welcome to FOR THE BUS GEEKS.
Ever wanted to make your own bus routes in any city you want? Felt like MetroDreamin was, well, mainly Metro related? Worry not, I had these thoughts and feelings too, so with the help of Map API, I bring you FOR THE BUS GEEKS - a successor to FOR THE STREET GEEKS - ROAD MAKER.

# Features
<li>Fully supported bus mapping system</li>
<li>Load and saving feature</li>
<li>Waypoint marking for buses</li>
<li>MTA inspired things mainly</li>
